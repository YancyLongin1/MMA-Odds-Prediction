<h1>MMA-Odds-Prediction</h1>

<h2>Description</h2>
Machine Learning project focused on predicting the outcomes of MMA fights using historical data. This project explores how betting odds, weight class, and other features influence fight outcomes. Additionally, it analyzes underdog wins in different matchups.
<br />

<h2>Dataset</h2>
The dataset includes the following information:
- <b>R_fighter</b>: Red corner fighter's name.  
- <b>B_fighter</b>: Blue corner fighter's name.  
- <b>R_odds</b>: Betting odds for the Red corner.  
- <b>B_odds</b>: Betting odds for the Blue corner.  
- <b>date</b>: Date of the fight.  
- <b>location</b>: Location of the fight.  
- <b>country</b>: Country where the fight took place.  
- <b>Winner</b>: Outcome of the fight (Red/Blue).  
- <b>weight_class</b>: Weight class of the fighters.  
- <b>gender</b>: Gender of the fighters.
